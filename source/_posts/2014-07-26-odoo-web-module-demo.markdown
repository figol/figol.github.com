<h3>项目简介</h3>
<pre>
web_scanner是一个典型的odoo web模块.
此模块包括了以下知识点:
1. web模块开发方法,下载安装即可.
2. dialog, blockUI, backbone, Qweb Template; 
3. odoo中, js调用python code的方法;
4. 一个简单的扫描枪应用实例;
</pre>
<h3>注</h3>
<pre>
本模块适用于具有一定经验的odoo程序员学习.
前提知识: jquery, backbone, qweb
</pre>
